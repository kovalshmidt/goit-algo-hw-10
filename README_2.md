Функція `quad` повертає значення інтеграла з дуже малою похибкою  
**2.6666666666666665 ± 2.9605947323337504e-14**, що можна вважати практично точним результатом.

### Результати методу Монте-Карло

- При `num_points = 1_000` отримано значення **2.728**, яке помітно відрізняється від точного результату.
- При `num_points = 100_000` отримано значення **2.6532**, похибка зменшується.
- При `num_points = 10_000_000` отримано значення **2.6668152**, яке є дуже близьким до аналітичного значення інтеграла.

### Висновок

Отримані результати підтверджують основну властивість методу Монте-Карло:  
**точність методу зростає зі збільшенням кількості випадкових точок**.  

Для малої кількості точок результат має значні випадкові відхилення, проте зі збільшенням кількості ітерацій значення інтеграла поступово збігається з точним аналітичним результатом.
